Created a python script where object (in dict format) and key are taken as user input.

Examples:
1. object = {"a":{"b":{"c":"d"}}} 
   key= a/b/c 
   value = d 

2. object = {"p":{"q":{"r":"s"}}} 
   key = p 
   value = {'q': {'r': 's'}}

3. object = {"x":{"y":{"z":"w"}}}
   key= x/y
   value = {'z': 'w'}
